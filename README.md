                                                                                 TouristPlanner

DESCRIPTION: 
TouristPlanner is a web-based application that helps users plan trips by exploring destinations, managing itineraries, and storing travel-related information in a structured way.

FEATURES:
1. User registration and login
2. View and manage tourist destinations
3. Create and update travel plans
4. REST APIs for flights and hotel related operations
5. Basic CRUD operations

TECH STACK: 
1. Backend: Java, Spring Boot, Spring Data JPA
2. Frontend: React.js
3. Database: MySQL
4. Tools: Postman

APPLICATION FLOW:
User sends requests from React UI
Spring Boot REST APIs handle business logic
JPA interacts with the database
Data is returned in JSON format
